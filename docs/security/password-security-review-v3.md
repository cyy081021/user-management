# 用户管理系统 — 密码安全审计报告 (v3)

## 密码策略

| 项目 | 要求 |
|------|------|
| 最小长度 | 12 位 |
| 大写字母 | ≥1 |
| 小写字母 | ≥1 |
| 数字 | ≥1 |
| 特殊字符 | ≥1 |
| 存储 | PBKDF2 哈希 (Werkzeug) |
| 来源 | 环境变量，源码无明文 |
| 占位拒绝 | "在此处填写"、placeholder 等 → 启动失败 |

## 启动校验流程

```
启动
 ├─ 读取 ADMIN_PASSWORD (环境变量)
 ├─ 读取 ALICE_PASSWORD   (环境变量)
 ├─ 检查是否为占位符/弱词 → 拒绝并退出
 ├─ 检查强度是否达标      → 拒绝并退出
 ├─ generate_password_hash() → USERS
 └─ 启动完成
```

## 认证流程

```
POST /login
 ├─ 防枚举延迟 (0.5~1.5s)
 ├─ 检查账号是否锁定 (Redis) → 已锁定则拒绝
 ├─ check_password_hash() 比对
 │   ├─ 成功 → 清除失败计数 → 登录
 │   └─ 失败 → 记录失败计数 (Redis)
 │              ├─ <5次 → 返回错误
 │              └─ ≥5次 → 锁定 15 分钟
 └─ 安全响应头 + CSRF + 速率限制
```
