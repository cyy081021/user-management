# 安全策略

## 报告安全漏洞

如果您发现了安全漏洞，请 **不要** 在 GitHub Issues 中公开报告。

请通过以下方式私下报告：

1. 在 GitHub 上创建一个 **Security Advisory**：
   https://github.com/cyy081021/user-management/security/advisories

2. 或发送邮件至项目维护者

## 范围

本项目是一个教学演示项目，但以下问题仍被视为安全缺陷：

- 认证绕过
- 敏感信息泄露
- CSRF/XSS/SSRF 防护缺失
- 依赖中的已知 CVE
- 会话劫持

## 响应时间

我们将在 7 天内确认收到报告，并在 30 天内发布修复版本。
